# JTBD — Minesweeper

## Назначение

Minesweeper помогает пользователю решать прикладную задачу в управляемом продуктовым контуре.

---

## Пользователи

- основные пользователи первой версии продукта;

---

## Проблема пользователя

У пользователя должна быть понятная отправная точка продукта без ручной пересборки базового контура.

---

## Задачи пользователя

1. Когда я начинаю работу с продуктом, то хочу получить понятный стартовый контур, чтобы быстрее перейти к предметному наполнению.
2. Когда я продолжаю работу над продуктом, то хочу видеть согласованный набор базовых документов, чтобы не терять контекст.

---

## Ценный результат для пользователя

Пользователь использует продукт, чтобы:
- быстрее перейти к предметной работе;
- удерживать продукт в понятном и проверяемом состоянии.
