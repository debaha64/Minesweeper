# JTBD — Minesweeper

## Назначение

Minesweeper помогает одному локальному игроку запустить классическую игру «Сапёр» на Python и сыграть партию.

---

## Пользователи

- один локальный игрок;

---

## Проблема пользователя

Пользователь хочет сыграть в «Сапёр» локально без установки дополнительных зависимостей и без настройки проекта.

---

## Задачи пользователя

1. Когда я открываю репозиторий, то хочу запустить игру одной командой, чтобы сразу начать партию.
2. Когда я играю, то хочу открывать клетки и ставить флаги, чтобы проходить поле по классическим правилам «Сапёра».
3. Когда партия завершается, то хочу видеть победу или поражение, чтобы понимать результат.

---

## Ценный результат для пользователя

Пользователь использует продукт, чтобы:
- сыграть локальную партию «Сапёра»;
- проверить первый playable pass без внешних зависимостей.
