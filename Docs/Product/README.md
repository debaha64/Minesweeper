# Product

## Назначение
Слой `Docs/Product/` описывает продукт в прикладных терминах первой версии.

## Состав слоя
- `Product_Passport.md` — паспорт созданного каркаса без домена `Profiles/*`.
- `JTBD.md` — пользовательские задачи и ценный результат.
- `PRD.md` — продуктовые требования первой версии.
- `Delivery.md` — краткая модель поставки продукта.
