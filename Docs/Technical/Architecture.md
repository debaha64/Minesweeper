# Architecture

`Minesweeper` разделяет игровую логику, desktop UI и продуктовый контур репозитория.

## Компоненты
- `minesweeper/core.py` — чистая игровая модель: поле, мины, флаги, открытие клеток, победа и поражение.
- `minesweeper/app.py` — `tkinter`-интерфейс desktop-приложения.
- `main.py` — точка запуска игры из корня репозитория.
- `tests/test_minesweeper_core.py` — unit-тесты игровой логики без графического окна.

## Решение
Игровая логика не зависит от `tkinter`, поэтому её можно проверять в headless-среде. UI только вызывает методы модели и перерисовывает кнопки поля.
