# Architecture

`Minesweeper` состоит из двух слоёв:

- `minesweeper.engine` — чистая доменная логика поля, мин, флагов и состояний игры;
- `minesweeper.app` — desktop GUI на `tkinter`, который отображает состояние и передаёт действия пользователя в engine.

Доменная логика не зависит от GUI и проверяется через `unittest`. GUI зависит только от стандартной библиотеки Python.
