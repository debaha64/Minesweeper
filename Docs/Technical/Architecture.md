# Architecture

`Minesweeper` разделяет игровую модель и desktop UI.

## Компоненты
- `minesweeper.game` — правила игры, состояние поля, мины, флаги, победа и поражение.
- `minesweeper.app` — Tkinter-интерфейс поверх игровой модели.
- `tests/test_game.py` — unit-тесты правил без запуска GUI.

## Решение
Первая версия использует только стандартную библиотеку Python. Это сохраняет локальный запуск простым и позволяет проверять игровую логику в headless-среде.
