# System_Invariants

## Инварианты
- продукт остаётся отдельным репозиторием вне дерева `BytePress`;
- human/agent entry contour не спорит с planning contour;
- подтверждённый discovery bootstrap является основанием первого предметного pass;
- product repo не превращается в полную копию `BytePress`.

## Игровые инварианты
- первое открытие клетки не должно приводить к поражению;
- флаг нельзя поставить на уже открытую клетку;
- флагированная клетка не открывается обычным действием открытия;
- победа наступает только после открытия всех безопасных клеток;
- поражение наступает после открытия клетки с миной.
