# System Invariants

## Продуктовые инварианты
- `AGENTS.md` остаётся точкой входа агента.
- `Docs/Discovery/Interview.md` владеет текущей истиной.
- `Plans/*` владеет этапами, задачами и pass.
- `Tools/*` владеет локальными проверками.

## Игровые инварианты
- Первая открытая клетка не содержит мину.
- Открытая клетка не может быть помечена флагом.
- Флаг не открывает клетку.
- Победа наступает после открытия всех безопасных клеток.
- Поражение наступает после открытия клетки с миной.
