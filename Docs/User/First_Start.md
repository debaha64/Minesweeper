# First Start

## Первый маршрут
1. Прочитать `../../README.md`.
2. Прочитать `Operating_Mode.md`.
3. Прочитать `../../Setup_Guide.md` и открыть task-ветку до первого writable action.
4. Проверить current stage/task/pass в `../../Plans/*`.
5. Прочитать `../Discovery/Interview.md` и проверить подтверждённую current truth.
6. Из корня продукта запустить игру: `python3 -m minesweeper`.
7. Для локальной проверки выполнить `python3 -m unittest`.
8. Из корня продукта выполнить `BYTEPRESS_ROOT=/path/to/BytePress scripts/dev-test.sh`, если нужен structural check через BytePress.
9. Если ранний product-start сорвался, выполнить `../../scripts/reset-product-start.sh`.

## Что дальше
- если нужен новый pass: `Pass_Request.md`;
- если нужен agent execution route: `../../AGENTS.md`.
