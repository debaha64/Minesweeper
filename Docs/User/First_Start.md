# First Start

## Первый маршрут
1. Прочитать `../../README.md`.
2. Прочитать `Operating_Mode.md`.
3. Прочитать `../../Setup_Guide.md` и открыть task-ветку до первого записываемого действия.
4. Проверить текущие этап, задачу и план в `../../Plans/*`.
5. Прочитать `../Discovery/Interview.md` и сверить подтверждённую текущую истину.
6. Запустить игру из корня продукта: `python3 -m minesweeper`.
7. Проверить игровую логику: `python3 -m unittest discover`.
8. Из корня продукта выполнить `python3 Tools/product_check.py --repo . --mode auto`.
9. Если нужен локальный smoke, выполнить `python3 Tools/product_bootstrap_smoke.py`.

## Что дальше
- если нужен новый pass: `Pass_Request.md`;
- если нужен маршрут исполнения агентом: `../../AGENTS.md`.
