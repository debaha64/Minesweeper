# First Start

## Первый маршрут
1. Убедиться, что доступен Python 3.11+ с `tkinter`.
2. Из корня репозитория выполнить `python3 main.py`.
3. Открывать клетки левой кнопкой мыши.
4. Ставить или снимать флаг правой кнопкой мыши, либо `Ctrl` + левая кнопка.
5. Победить, открыв все безопасные клетки; проигрыш наступает при открытии мины.
6. Для проверки игровой логики выполнить `python3 -m unittest`.
7. Для проверки продуктового каркаса выполнить `python3 Tools/product_check.py --repo . --mode auto`.
