# Usage Scenarios

## Сценарий 1. Войти в продуктовый репозиторий
- Прочитать `../../README.md` и `First_Start.md`.
- Подтвердить текущую истину в `../../Docs/Discovery/Interview.md` до предметных правок.
- Подготовить среду по `../../Setup_Guide.md`.
- Проверить структурный контур через `Tools/product_check.py`.

## Сценарий 2. Сыграть партию
- Из корня репозитория выполнить `python3 -m minesweeper`.
- Открывать клетки левой кнопкой мыши.
- Помечать предполагаемые мины правой кнопкой мыши.
- Завершить партию победой или поражением и при необходимости нажать `Restart`.

## Сценарий 3. Запустить новый pass с агентом
- Найти текущие этап, задачу и план в `../../Plans/*`.
- Сформулировать pass через `Pass_Request.md`.
- Направить агента на работу через contracts продукта.

## Сценарий 4. Работать с продуктовым контуром знания
- `Docs/Product/*` — прикладная рамка продукта.
- `Docs/Technical/*` — стартовый технический контур.
- `Logs/*` — факты и доказательства качества.
