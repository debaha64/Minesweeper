# Usage Scenarios

## Сценарий 1. Играть в Сапёр
- Запустить `python3 main.py`.
- Открывать клетки левой кнопкой мыши.
- Использовать числа соседних мин для выбора следующего хода.
- Ставить флаги правой кнопкой мыши или `Ctrl` + левая кнопка.
- Завершить партию победой или поражением.

## Сценарий 2. Запустить новый pass с агентом
- Найти текущие этап, задачу и план в `../../Plans/*`.
- Сформулировать pass через `Pass_Request.md`.
- Направить агента на работу через contracts продукта.

## Сценарий 3. Работать с продуктовым контуром знания
- `Docs/Product/*` — прикладная рамка продукта.
- `Docs/Technical/*` — стартовый технический контур.
- `Logs/*` — факты и доказательства качества.
