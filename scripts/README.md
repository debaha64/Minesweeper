# scripts

`scripts/*` — project entry scripts replicated product repo.

- `dev-up.sh` — placeholder старта локального product contour.
- `dev-down.sh` — placeholder остановки локального contour.
- `dev-test.sh` — structural check route через `BYTEPRESS_ROOT`.
- `integration-smoke.sh` — controlled integration handoff route через `BYTEPRESS_ROOT` с report artifact в `Runtime/Integration_Smoke_Report.json`.
