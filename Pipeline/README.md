# Pipeline

`Pipeline/*` хранит лёгкий локальный процессный контур продукта и владеет рабочим процессом агента.

## Состав
- `Phases.md` — минимальные фазы product pass.
- `Workflows.md` — основной путь, первый product-start, предметный проход, уровни проверок, журнальное закрытие и PR через `gh`.
- `Gates.md` — обязательные переходы и проверки.

## Граница
`AGENTS.md` направляет сюда, но не подменяет этот процессный контур.
