# Gates

## Current truth gate
- `Статус_текущей_истины: Не_подтверждена` удерживает продукт в аналитическом контуре.
- Первое записываемое действие требует task-ветку.
- Fresh state проверяется локальным `Tools/product_check.py`.

## Dependency gate
- стек и зависимости требуют явного источника;
- GUI-модуль вроде `tkinter` требует проверки доступности;
- непроверенный GUI-запуск фиксируется как `не проверено`.

## PR gate
- PR создаётся через `gh` после final push;
- GitHub connector не используется для создания PR.
