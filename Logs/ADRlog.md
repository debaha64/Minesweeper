# ADRlog

## Индекс

### ADR-000001 — Стек первого прохода: Python, стандартная библиотека, без GUI
ID: ADR-000001
Дата: 2026-05-02
Связи: PLAN-000002, BACK-000002
Статус: Утверждено
Контекст: Пользователь подтвердил стек логики — Python стандартная библиотека. GUI (tkinter) относится к следующему проходу.
Решение: Первый проход — библиотека логики на Python без внешних зависимостей. GUI на tkinter — отдельный проход.
Последствия: Логика игры отделена от GUI, что обеспечивает переиспользуемость и тестируемость.
