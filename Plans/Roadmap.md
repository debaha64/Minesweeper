## ROAD-000002 — Реализовать библиотеку логики игры Сапёр
ID: ROAD-000002
Этап: Реализовать библиотеку логики игры Сапёр
Статус: В_работе
Связи: BACK-000002
Источник: PLAN-000002
Дата_создания: 2026-05-02
Дата_изменения: 2026-05-02
Цель: Реализовать библиотеку логики Сапёра как первый предметный проход: создание поля, расстановка мин, открытие клеток, определение победы/поражения. Без GUI.
Зависимости: ROAD-000001
Связанные_backlog: BACK-000002

### Описание
Первый предметный проход: библиотека логики игры Сапёр на Python (стандартная библиотека). Граница: один сценарий — новая игра → расстановка мин → открытие клеток → победа/поражение. Вне границы: сетевая игра, GUI, таблица рекордов.
